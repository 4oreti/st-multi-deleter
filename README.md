# 🗑️ SillyTavern 批量删除与搬家助手 (ST-Multi-Deleter)

![SillyTavern Extension](https://img.shields.io/badge/SillyTavern-Extension-blue.svg)
![Version](https://img.shields.io/badge/Version-1.0.0-success.svg)

一款为 SillyTavern（酒馆）量身定制的高级聊天记录管理插件。告别繁琐的单条删除和误删风险，以超级高效的方式管理你的聊天记录！

## ✨ 核心特色功能 (Features)

本插件彻底重构了酒馆的删楼交互体验，带来了以下独家高级功能：

*   🃏 **「卡片式队列」防呆预览**
    *   告别枯燥的列表和丑陋的复选框！所有选中的待处理信息都会化作精致的“任务卡片”。
    *   **一键豁免**：在预览中发现错选？点击卡片上的“撤出”按钮，卡片会伴随丝滑的动画飞出队列，完美防止误删！
*   🚚 **「一键搬家」功能**
    *   想要把某段精彩的剧情或跑偏的支线独立出来？选中楼层后点击“搬家”！
    *   插件会自动将选中的记录打包为 **SillyTavern 官方标准的 `.jsonl` 聊天文件** 并下载，同时将它们从当前聊天中移除。你可以去任何角色那里使用“导入聊天记录”无缝衔接！
*   📱 **全端适配 & 高级多选**
    *   **范围输入**：手机端福音！直接输入 `0-12, 15, 18-20`，点击“范围选中”，瞬间精准捕获目标楼层。
    *   **Shift 连选**：电脑端支持像操作本地文件一样，按住 `Shift` 键点击气泡，瞬间选中大段连续楼层。
    *   **向下全选**：一键选中当前楼层及以下的所有内容，一刀切掉重开的最佳伴侣。
*   💊 **贴心的后悔药机制**
    *   执行删除时，可勾选“删除前下载TXT备份”。插件会在执行摧毁前，自动为你生成一份如剧本般排版精美的本地文本备份，再也不怕“手残”！
*   🎨 **无感防触碰 UI**
    *   进入管理模式后，聊天气泡会覆盖专属交互层（变暗+红框打勾），彻底隔绝原生的编辑、滑动等按钮，绝不误触！

## 📦 安装方法 (Installation)

由于本插件完全遵循 SillyTavern 官方模块化规范，你可以直接通过酒馆内置的拓展管理器一键安装：

1. 打开你的 SillyTavern。
2. 点击顶部菜单栏的 **扩展 (Extensions)** 图标 🧩。
3. 点击 **安装扩展 (Install Extension)**。
4. 在输入框中粘贴本仓库的 GitHub 链接：
   `https://github.com/oreti/st-multi-deleter` *(上传后记得把这里换成你真实的链接哦！)*
5. 点击 **Save (保存/安装)**，等待几秒钟，安装完成后刷新页面 (F5) 即可使用！

## 💻 使用说明 (How to Use)

1. 点击顶部菜单 🧩 扩展里的 **批量删除信息** 工具，进入管理模式。
2. **选择楼层**：
   * 鼠标/手指直接点击想要处理的聊天气泡。
   * 使用底部控制面板的“全选”、“向下全选”或输入数字范围。
3. **预览与操作**：
   * 点击底部红色的 **删除** 按钮，打开卡片队列。
   * 检查无误后，选择点击 **确认删除** 彻底清空，或者点击 **搬家** 导出为新存档。
4. 点击 **取消** 随时退出管理模式。

## ⚠️ 注意事项

* 本插件直接调用酒馆底层原生 `deleteMessage` API，数据操作 100% 安全且符合官方规范。
* 搬家导出的 `.jsonl` 文件包含了时间戳和发送者等完整元数据，完美兼容酒馆原生导入系统。
* 虽然有防呆和备份机制，但在进行超大规模的删除前，仍建议对你的酒馆数据做好日常备份。
